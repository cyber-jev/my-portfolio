## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/cyber-jev/my-portfolio.git

# Navigate into the project
cd my-portfolio

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
